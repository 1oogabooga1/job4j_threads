# job4j_threads

Этот проект предназначен для изучения основ многопоточности в Java.

В проекте рассмотрено следующее:
1. Threads
2. Общие ресурсы
3. Синхронизация ресурсов
4. Wait, Notify, NotifyAll
5. Non Blocking Algorithm
6. Пулы
7. NIO и многопоточность
